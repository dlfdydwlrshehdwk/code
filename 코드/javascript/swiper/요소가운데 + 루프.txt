예시는 3개를 보여주는 옵션 가운데하나 왼쪽에 반쪽짜리하나 오른쪽에 반쪽짜리 하나
let swiper = new Swiper('.mySwiper', {
        slidesPerView: 2, // 한번에 보여줄 개수
        slidesPerGroup: 1, // 한번 스와이프시 넘어갈 개수
        spaceBetween: 100, // 요소간 간격
        loop: true, // 루프할건지?
        centeredSlides: true , // 요소 중앙부터
        loopedSlides: 2, // 이걸 설정해줘야 1번째 요소가 가운데로 설정되게끔보임 설정하지 않으면 왼쪽에 반쪽짜리로 있음
})

주의점 3개를 무한으로 돌리려면 3개로는 비어있는 공간이 보인다.
고로 clone을 해서 요소를 2배로만들거나
하드코딩으로 같은 슬라이드를 2배로넣거나 하는 등의 행동을 해야함.

ex ) 3개로 무한으로 돌리려면 최소 6개가 필요함
