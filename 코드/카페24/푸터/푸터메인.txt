모듈 : module="Layout_footer"

Class 반환 코드값 : xans-element- xans-layout xans-layout-footer

- 사용 가능 변수 -
{$bottom_logo} : 화면 하단 쇼핑몰 로고
{$mall_name} : 상점명
{$company_name} : 법인명 (상호)
{$phone} : 대표전화
{$fax} : 대표팩스
{$email} : 대표메일
{$mall_zipcode} : 사업장주소 - 우편번호
{$mall_addr1} : 사업장주소 1 ( 시군구 )
{$mall_addr2} : 사업장주소 2 ( 상세주소 )
{$president_name} : 대표자 성명
{$company_regno} : 사업자 등록번호
{$network_regno} : 통신판매업 신고
{$runtime} : 운영시간
{$cpo_name} : 개인정보보호 책임자 성명
{$cpo_email} : 개인정보보호 책임자 메일
{$biz_no_link} : 사업자정보확인 링크
{$action_biz_info} : 사업자정보확인 링크 (onclick)
